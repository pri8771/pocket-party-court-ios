# Technical PRD

## Objective
Ship an offline-first iOS 17+ SwiftUI party game scaffold for local case setup, reveal, timed arguments, voting, and verdict sharing.

## Requirements
- SwiftUI UI architecture
- SwiftData persistence for decks, cases, sessions, and players
- No third-party dependencies
- No backend, login, cloud sync, or network requirement
- Starter content bundled in app resources

## Initial Screens
Home, game setup, case reveal, timer, voting, verdict, deck list, and deck detail.
